
# 📊  Portfolio Tracker

** Portfolio Tracker** is a Windows desktop application built with **.NET MAUI** that helps users track and analyze their investment performance on XTB accounts.  
The app allows users to **import data directly from XTB-generated files**, view profit and loss across multiple accounts, and compare their performance against the **S&P 500 index**.

---

## 🚀 Features

- 📂 Import and process trading data from XTB export files  
- 💰 Track earnings across multiple investment accounts  
- 📈 View total and account-level profit/loss summaries  
- 📊 Compare portfolio results to S&P 500 performance  
- 🧮 Aggregate data for an overall profit summary  
- 🪄 Clean and responsive UI built with .NET MAUI and XAML

## 🛠️ Technologies Used

- **C# 12 / .NET 8**
- **.NET MAUI (Windows platform)**  
- **XAML** UI definitions
- **Visual Studio 2022**
- **LINQ** for data aggregation and filtering
- **LiveCharts2** — for creating interactive financial charts
- **ClosedXML** — for importing and reading Excel files exported from XTB
